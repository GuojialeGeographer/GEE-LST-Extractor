### 2. 数据缩放说明 ✅

**问题**: 用户不知道如何处理提取的原始数据值
**解决方案**: 添加详细的数据缩放说明

**添加内容**:

#### 在代码中添加说明（单元格10）
```python
# 转换单位
# MODIS LST数据定标说明：
# - 原始值需要乘以缩放因子0.02
# - 然后减去273.15转换为摄氏度
# - 公式: LST(°C) = LST_raw × 0.02 - 273.15
```

#### 在快速参考中添加缩放表格（单元格17）

| 数据类型 | 缩放因子 | 公式 | 说明 |
|---------|---------|------|------|
| LST | 0.02 | `value × 0.02 - 273.15` | 转换为摄氏度 |
| NDVI | 0.0001 | `value × 0.0001` | 范围-1到1 |
| EVI | 0.0001 | `value × 0.0001` | 范围-1到1 |
| Albedo | 0.001 | `value × 0.001` | 0到1 |

**示例代码**:
```python
# LST缩放
df['LST_Celsius'] = df['LST_Day_1km'] * 0.02 - 273.15

# NDVI缩放
df['NDVI_scaled'] = df['NDVI'] * 0.0001

# EVI缩放
df['EVI_scaled'] = df['EVI'] * 0.0001

# Albedo缩放
df['Albedo_scaled'] = df['Albedo_WSA_shortwave'] * 0.001
```

## 💡 使用建议

### 对于新用户

1. **提取数据后，记得应用缩放因子**
   ```python
   # 提取原始数据
   result = extractor.extract_data(...)

   # 应用缩放
   result['LST_Celsius'] = result['LST_Day_1km'] * 0.02 - 273.15
   result['NDVI_scaled'] = result['NDVI'] * 0.0001
   ```

2. **检查数据范围**
   - LST: 通常在-30°C到50°C之间
   - NDVI: 范围-1到1，植被区域通常0.2-0.9
   - EVI: 范围-1到1
   - Albedo: 范围0到1

3. **处理缺失值**
   ```python
   # 检查缺失值
   print(result.isna().sum())

   # 填充缺失值
   result = result.fillna(method='ffill')
   ```
